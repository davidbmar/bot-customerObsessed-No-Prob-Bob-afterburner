`★ Insight ─────────────────────────────────────`
- **Theme via CSS custom properties**: Because the existing code already used `var(--bg)`, `var(--text)`, etc. everywhere, adding a light theme was just defining an `html.light` class with different values — zero changes to individual component styles. This is the power of CSS custom properties for theming.
- **Search with snippet extraction**: Rather than just filtering, the search extracts a context snippet around the match and highlights it with `<mark>`, giving the user visual confirmation of *where* the keyword appeared in the conversation.
