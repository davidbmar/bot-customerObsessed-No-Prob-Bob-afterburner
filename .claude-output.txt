`★ Insight ─────────────────────────────────────`
The root cause of B-028 was a classic "wiring gap" — `BotConfig.load()` was called in `main()` and used to configure the Gateway's LLM/personality, but the config object itself was never passed through. The Gateway already accepted a `config` parameter (line 51), so the fix was a single keyword argument addition. The response format change from `["slug"]` to `[{"slug": "...", "name": "..."}]` ensures the frontend dropdown can display human-readable names.
